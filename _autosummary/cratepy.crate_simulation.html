Perform CRATE simulation.

Parameters:

  • arg_input_file_path (str) – Input data file path provided as input.

  • arg_discret_file_dir (str, default=None) – Spatial discretization file directory path provided as input.

  • is_null_stdout (bool, default=False) – Suppress execution output to stdout.
